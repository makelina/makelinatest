## 🚀 Запуск автотестов (Playwright)

1. Перейдите в папку с тестами:
   ```bash
   cd playwright-test
   ```
2. Установите зависимости:
   ```bash
   npm install
   npx playwright install
   ```
3. Запустите тесты:
   ```bash
   npm test
   ```
